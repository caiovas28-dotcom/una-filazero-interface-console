# una-filazero-interface-console

Fila Zero - Sistema de Login

Aplicação de console em C# que simula um sistema básico de autenticação de usuário.

Descrição

O programa exibe uma interface simples no console, solicita o nome de usuário e a senha, e valida as credenciais informadas.

Funcionalidades
Exibição de tela inicial no console
Entrada de nome de usuário e senha
Validação de login com dados fixos
Mensagem de sucesso ou erro
Credenciais de acesso
Usuário: admin
Senha: senha123

Como executar:

Salve o código em um arquivo chamado Program.cs
Abra o terminal na pasta do projeto

Execute o comando:
dotnet run
Estrutura do código
DesenharTela()
Responsável por exibir o cabeçalho da aplicação.
ExecutarLogin()
Solicita os dados do usuário e realiza a validação.
